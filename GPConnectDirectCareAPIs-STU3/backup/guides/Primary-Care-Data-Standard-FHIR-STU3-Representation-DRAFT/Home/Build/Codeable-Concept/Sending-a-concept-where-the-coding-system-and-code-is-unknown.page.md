## {{page-title}}

In the exceptional case that the code is not known, then the code.text element can be
populated without the need to populate a coding element.

```xml
<code>
    <text>Myocardial infarction</text>
</code>
```

```json
{
    "code": {
        "text": "Myocardial infarction"
    }
}
```
