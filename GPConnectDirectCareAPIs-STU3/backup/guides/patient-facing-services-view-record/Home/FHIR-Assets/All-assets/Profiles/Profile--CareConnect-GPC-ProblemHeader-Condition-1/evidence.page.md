<h5><ins>Guidance</ins></h5>

This `stage` element does not need to be populated.

```xml
<evidence>

            <code value="1148601009" />
            <display value="Able to thrive (finding)" />
        </coding>
    </code>
    <detail value="reference-resource-008912jd" />
</evidence>
```
