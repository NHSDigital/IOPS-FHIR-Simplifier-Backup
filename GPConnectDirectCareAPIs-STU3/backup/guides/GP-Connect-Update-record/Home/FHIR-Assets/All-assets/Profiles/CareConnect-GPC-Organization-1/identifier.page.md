## {{page-title}}

<h5><ins>Guidance</ins></h5>

The `identifier` element **SHOULD** be populated with the ODS code of the patients registered GP practice

<h5><ins>Example</ins></h5>

```xml
<identifier>
    <system value="https://fhir.nhs.uk/Id/ods-organization-code" />
    <value value="ODS1234" />
</identifier>
```
