## {{page-title}}

The code system for the NHS Dictionary of Medicines and Devices (dm+d) is `https://dmd.nhs.uk`.

e.g.
```xml
<medicationCodeableConcept>
     <coding>
        <system value="https://dmd.nhs.uk" />
        <code value="317972000" />
        <display value="Furosemide 40mg tablets" />
     </coding>
</medicationCodeableConcept>
```
