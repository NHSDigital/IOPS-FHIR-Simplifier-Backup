## StructureDefinition-UKCore-Provenance

Defines the UK Core constraints and extensions on the provenance resource for the minimal set of data to query and retrieve generic provenance information.

# Profile Purpose #
The Provenance resource tracks information about the activity that created, revised, deleted, or signed a version of a resource, describing the entities and agents involved.
