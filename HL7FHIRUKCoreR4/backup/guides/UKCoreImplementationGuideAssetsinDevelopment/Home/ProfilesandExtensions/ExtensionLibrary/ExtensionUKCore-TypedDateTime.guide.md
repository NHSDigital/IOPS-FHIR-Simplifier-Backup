## StructureDefinition Extension-UKCore-TypedDateTime
This supports the exchange of dates and/or times. The extension date and/or time is carried along with a code to identify the type of date and/or time.


### Purpose
This extension extends the Encounter and the Communication resources to support the exchange of information describing the specific business related date and/or time which is currently not supported by the FHIR standard.
